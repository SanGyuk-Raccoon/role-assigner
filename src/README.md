# 역할 뽑기 (Role Assigner)

파티 게임을 위한 역할 배정 도구입니다.

## 기능

### 핵심 기능
- **공개 모드**: 한 화면에서 모든 역할 공개 (오프라인)
- **비공개 모드**: 각자 기기에서 본인 역할만 확인 (실시간 멀티플레이어)
- **마니또 모드**: 시크릿 산타 1:1 배정 (자기 자신 배정 방지)
- **커스텀 역할**: 원하는 역할 세트 직접 구성

### 역할 수 설정
- 숫자 지정: 해당 역할을 정확히 그 수만큼 배정
- 0명 설정: 남은 인원을 해당 역할로 자동 배정

## 사용법

### 공개 모드 (오프라인)
1. 역할 설정 (이름 + 인원수)
2. 참가자 이름 입력
3. 셔플 후 화면에서 모두 확인

### 비공개 모드 (멀티플레이어)
1. 호스트가 방 생성
2. 6자리 코드 공유
3. 참가자들이 각자 기기로 접속
4. 호스트가 역할 배정
5. 각자 본인 역할만 확인

### 마니또 모드
1. 방 생성 시 마니또 모드 활성화
2. 참가자들 입장
3. 역할 배정 시 자동으로 1:1 매칭
4. 각자 누구에게 선물할지 확인

## 지원 게임

- 마피아 게임
- 라이어 게임
- 마니또/시크릿 산타
- 원카드 역할 배정
- 기타 커스텀 파티 게임

## 기술 정보

- 방 코드: 6자리 (혼동 방지를 위해 I/O/0/1 제외)
- 최대 참가자: 20명 (비공개 모드)
- 방 유효기간: 24시간
- 셔플 알고리즘: Fisher-Yates
- 마니또 알고리즘: Sattolo (자기 자신 배정 불가능 보장)
